# 002: 変数と定数

## このレッスンで学ぶこと

- `var` による変数宣言
- `:=` による短縮宣言（型推論）
- `const` による定数宣言
- Go の基本型（`int`, `float64`, `string`, `bool`）
- ゼロ値

## コード解説

### var 宣言

```go
var name string = "太郎"
var age int = 25
```

型を明示して変数を宣言する方法。パッケージレベル（関数の外）でも使える。

### 短縮宣言 `:=`

```go
city := "東京"
score := 98.5
```

**関数内でのみ使える**。型は右辺から自動推論される。
最もよく使う宣言方法。

### ゼロ値

Goでは変数を初期化しなくても**ゼロ値**が入る。

| 型 | ゼロ値 |
|---|-------|
| `int` | `0` |
| `float64` | `0.0` |
| `string` | `""` (空文字) |
| `bool` | `false` |

### 定数

```go
const pi = 3.14159
```

`const` は変更不可。`:=` は使えない。

## 注意点

- **未使用の変数はコンパイルエラー**になる（Goの特徴）
- 保存時に VSCode が未使用変数を警告してくれる

## やってみよう

1. 自分の名前・年齢・好きな食べ物を変数に入れて出力してみよう
2. `var x int` だけ宣言して `fmt.Println(x)` するとどうなるか確認しよう
3. 宣言した変数を使わずに保存するとどうなるか試してみよう
