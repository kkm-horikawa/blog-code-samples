# 006: マップ

## このレッスンで学ぶこと

- マップの作成方法（リテラル, `make`）
- 要素の追加・取得・削除
- 存在チェック（カンマOKイディオム）
- マップの走査

## コード解説

### マップの作成

```go
// リテラルで作成
scores := map[string]int{
    "国語": 80,
    "数学": 95,
}

// make で空のマップを作成
m := make(map[string]int)
```

`map[キーの型]値の型` がマップの型。

### 要素の操作

```go
m["英語"] = 88      // 追加・更新
score := m["国語"]   // 取得
delete(m, "数学")    // 削除
```

### 存在チェック（カンマOKイディオム）

```go
value, ok := m["理科"]
if ok {
    fmt.Println("ある:", value)
} else {
    fmt.Println("ない")
}
```

存在しないキーを取得するとゼロ値が返る。
「値が0」なのか「キーがない」のかを区別するには `ok` を使う。

### 走査

```go
for key, value := range m {
    fmt.Printf("%s: %d\n", key, value)
}
```

**走査順序は毎回ランダム**（Go の仕様）。

## 注意点

- マップのゼロ値は `nil`。`nil` マップに書き込むとパニックする
- `make` かリテラルで初期化してから使うこと
- スライスと同様、マップも参照型
